# Hero Split Media Pattern

Two-column hero with message + CTA on one side and product media on the other.

## Structure

- Left: headline, subheadline, primary CTA, optional secondary CTA
- Right: product screenshot, mockup, or short looped demo

## Responsive Behavior

- Mobile: stack content then media
- Desktop: side-by-side with balanced visual weight

## Conversion Guidance

- Keep one dominant CTA action.
- Add compact proof cue under CTA (e.g., trust badges or adoption count).
- Ensure hero content remains clear even if media fails to load.
