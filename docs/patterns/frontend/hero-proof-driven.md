# Hero Proof-Driven Pattern

Hero variant that prioritizes trust signals close to the primary CTA.

## Recommended Elements

- Value proposition headline
- Primary CTA
- Compact trust row directly below CTA:
  - customer logos
  - star rating
  - short testimonial quote

## Notes

- Keep trust claims specific and verifiable.
- Avoid crowded proof stacks; one strong proof row is usually enough.
- Use descriptive alt text when logos communicate identity.
