# CTA Proof Stack

Reusable CTA layout that pairs action buttons with concise proof indicators.

## Layout

- CTA headline
- One primary action button
- Optional secondary action link
- Proof row below actions:
  - "No credit card"
  - "Trusted by X teams"
  - security or uptime badge

## Usage Guidance

- Keep proof row short and scannable.
- Ensure CTA contrast and focus states remain strong.
- Reuse across hero, mid-page, and end-page sections.
