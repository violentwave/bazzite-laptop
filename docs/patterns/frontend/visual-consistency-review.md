# Visual Consistency Review

Run this review to catch UI drift before handoff.

## Required Checks

- Typography scale is consistent across sections
- Spacing rhythm is consistent (no random jumps)
- Buttons and links share interaction states
- Color usage follows the chosen token palette
- Card/list/grid styles remain coherent

## Evidence

- Provide screenshots of at least three sections that demonstrate consistent styling.
- Add notes for any approved intentional deviations.
